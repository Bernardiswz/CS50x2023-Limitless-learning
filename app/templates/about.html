Overview

The website provides the user with resources and tools to two different learning and practicing

techniques. Those being the Pomodoro technique and the Flashcards technique. Aswell as a simple

history of user actions to track progress of their use of these tools.


It serves as an facilitator to engage in these techniques by eliminating the need of setting up a

timer manually aswell as writing the flashcards yourself. It mainly provides their core aspects,

serving as a building block, but it has room for improvements and more features.

How to take the most of these features?

It's advisable that the user is familiar with both of these techniques and for such there is many

free resources available online, not only on these specific techniques but in learning as a whole.

Some of them include the coursera courses but not limited to those:


  • "Learning how to Learn"


  • "Mindshift"
